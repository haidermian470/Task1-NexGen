# NexGen Solutions Website

This is my first HTML and CSS project. I built a fictional tech startup website called NexGen Solutions as part of my web development learning journey.

## What I Built
A fully responsive multi-section webpage including navigation, hero section, services, about, stats, testimonials, contact form and footer.

## Technologies Used
- HTML5
- CSS3
- Flexbox
- Media Queries
- CSS Animations
- Google Fonts
- CSS Variables

## What I Learned
- How to structure a complete webpage with semantic HTML
- How to style and layout using CSS Flexbox
- How to make a website responsive for mobile screens
- How to use CSS variables for consistent styling
- How to add hover animations and transitions
